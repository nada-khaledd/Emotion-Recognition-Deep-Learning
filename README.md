# Emotion Recognition using Deep Learning

This project implements a facial emotion recognition system using
MobileNetV2 and Transfer Learning.

## Emotions
- Angry
- Happy
- Neutral
- Sad
- Surprise

## Technologies
- Python
- TensorFlow / Keras
- OpenCV
- MobileNetV2

## Features
- Data augmentation
- Multi-stage fine-tuning
- Class imbalance handling
- High accuracy emotion classification
